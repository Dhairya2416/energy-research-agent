# Energy Research Agent

## Setup

1. Install dependencies:
   - `pip install -r requirements.txt`
2. Set environment variables:
   - `GROQ_API_KEY`
   - `TAVILY_API_KEY`
   - Optional: `LLM_MODEL` (default: `groq/llama-3.1-8b-instant`)

## Run

- Streamlit UI:
  - `streamlit run app.py`
- CLI:
  - `python main.py`
